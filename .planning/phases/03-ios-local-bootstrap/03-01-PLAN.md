<task type="auto">
  <name>Task 2: Bootstrap Capacitor iOS + sync de la web</name>
  <files>ios-app/ios/, ios-app/package-lock.json, ios-app/node_modules/</files>
  <action>
```bash
cd ios-app
npm install                        # dependencies Capacitor
npm run build:web                  # ejecuta scripts/sync-web.sh (copia ../index.html, ../img/, ../js/ → www/)
npx cap add ios                    # genera proyecto Xcode, corre pod install automático
npx cap sync ios                   # sincroniza www/ + plugins
```

Si `pod install` falla con "repo out of date", correr `pod repo update` y reintentar.

Evitar: no editar a mano nada dentro de `ios-app/ios/` todavía; ese directorio se regenera parcialmente en cada `cap sync`.
  </action>

  <done>`ios-app/ios/App/App.xcworkspace` existe. `npx cap doctor ios` no reporta errores.</done>

<task type="auto">
  <name>Task 3: Generar iconos + splash screens desde el logo</name>
  <files>ios-app/resources/icon.png, ios-app/resources/splash.png, ios-app/ios/App/App/Assets.xcassets/</files>
  <action>
1. Instalar `@capacitor/assets` si no está en el package.json: `cd ios-app && npm install -D @capacitor/assets`.

2. Generar `resources/icon.png` 1024×1024 desde `img/logo-buba.jpg`:

```bash
cd ios-app
mkdir -p resources
# Recortar el logo cuadrado centrado y componerlo sobre fondo cream FAF7F4
magick ../img/logo-buba.jpg -resize 768x768^ -gravity center -extent 768x768 \
  \( -size 1024x1024 xc:'#FAF7F4' \) +swap -gravity center -composite \
  resources/icon.png
```

3. Generar `resources/splash.png` 2732×2732 con logo centrado más pequeño:

```bash
magick ../img/logo-buba.jpg -resize 400x400^ -gravity center -extent 400x400 \
  \( -size 2732x2732 xc:'#FAF7F4' \) +swap -gravity center -composite \
  resources/splash.png
```

4. Generar todos los tamaños iOS con `@capacitor/assets`:

```bash
npx capacitor-assets generate --ios
```

Esto puebla `ios-app/ios/App/App/Assets.xcassets/AppIcon.appiconset/` y `Splash.imageset/`.
  </action>
  <verify>
```bash
ls ios-app/resources/*.png | wc -l  # >= 2
ls ios-app/ios/App/App/Assets.xcassets/AppIcon.appiconset/ | wc -l  # varios PNG generados
```
  </verify>
  <done>Icono 1024² y splash 2732² existen en `resources/`. Xcode ve los iconos en `AppIcon.appiconset/`.</done>
</task>
